## Como usar este proyecto

### Para el docente

1. Clona el repositorio en tu PC
2. Abre cada carpeta en orden (01, 02, 03...)
3. Ejecuta `teoria_y_ejemplos.py` para ver los ejemplos paso a paso
4. Asigna `ejercicios_propuestos.py` como tarea de practica

### Para el estudiante

1. Empieza por `01_funciones_sueltas/teoria_y_ejemplos.py` y leelo de arriba a abajo
2. Ejecutalo con `python3 teoria_y_ejemplos.py` para ver la salida
3. Cuando entiendas los ejemplos, abre `ejercicios_propuestos.py` y resuelvelos
4. Avanza al siguiente tema (02, 03, ...)
